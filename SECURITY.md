# Security Policy

Supported Versions

| Version | Supported |
|---------|-----------|
| v1.x    | ✅        |
| v0.x    | ❌        |

Reporting Vulnerabilities

Please do not open public GitHub issues for security vulnerabilities.

Instead, contact the maintainers privately.

Provide:

- Version
- Steps to reproduce
- Expected behavior
- Actual behavior
- Logs
